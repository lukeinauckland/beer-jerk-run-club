# V1 Launch Scope

## Launch Purpose

Help someone searching for a run club in Auckland quickly answer:

- What is this?
- Where is it?
- When is the next run?
- What does it cost?
- What pace is it?
- What happens after?
- Where do I follow or join?

## Included

- Homepage.
- New runner guide.
- Schedule page.
- Routes page.
- Afters page.
- 404 page.
- Static HTML content for search engines.
- JSON-LD structured data.
- `robots.txt`.
- `sitemap.xml`.
- `llms.txt`.
- `facts.json`.
- Responsive images.
- Vercel config.
- GitHub check workflow.

## Excluded On Purpose

- Streak Board.
- Half Marathon feature.
- Half Marathon ticket link.
- Newsletter signup.
- GPX route maps.
- Exact Christmas break dates.
- Real-time Instagram feed.
- CMS.

## Why

The launch site should be simple, findable and easy to update. Extra modules can come later once the core site is live and people can actually discover the club.
