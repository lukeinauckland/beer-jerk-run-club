## Search Console Notes

Use Google Search Console URL Inspection on each public page.

Request indexing in this order:

1. Homepage.
2. New runners.
3. Schedule.
4. Routes.
5. Afters.

## Rich Results Test

Run this on the homepage:

```text
https://search.google.com/test/rich-results
```

Expected structured data on the homepage:

- WebSite
- SportsOrganization
- SportsActivityLocation
- SportsEvent
- Schedule
- FAQPage

Warnings are possible. A warning is not automatically a launch blocker. Errors on core identity, event date or location are launch blockers.

## Manual Search Terms

Check branded terms first:

```text
Beer Jerk Run Club
Beer Jerk Run Club Auckland
BJRC Auckland
```

Then broad terms:

```text
Auckland run club
run club Auckland
social run club Auckland
Monday run club Auckland
Eden Terrace run club
free run club Auckland
```

Broad terms need time and backlinks. Do not panic if they lag at first.
