# Content Editing

Most content lives in:

```text
site.json
```

Edit that file, then run:

```sh
npm run check
```

## Common Updates

Beer offer:

```json
"beerDeal": "$10 beers for runners after every run."
```

Christmas wording:

```json
"christmasRule": "Christmas break dates are announced on Instagram each summer."
```

Public holidays:

```json
"publicHolidays": {
  "2026-10-26": "Labour Day"
}
```

Route rotation starts from:

```json
"anchorMonday": "2026-06-08",
"anchorRoute": "Beer Mile",
"rotation": ["Beer Mile", "Grange Hill", "Maungawhau"]
```

## Do Not Add For Launch

- Meetup as a public link.
- Half Marathon ticket link.
- Fake streak names.
- Fabricated club history.
- Claims about pace, cancellation history or member numbers unless Luke confirms them.
