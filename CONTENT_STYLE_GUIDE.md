# Content Style Guide

Use this when changing copy.

## Voice

- Direct.
- Warm.
- Useful.
- No self-congratulation.
- No fake lore.
- No invented stats.
- No "community" waffle unless it says something specific.
- No em dashes.

## Locked Phrases

Use exactly:

```text
People. Over. Pace.
$10 beers for runners after every run.
Free. Just turn up.
```

## Good BJRC Copy

```text
Bag drop from 5pm. We run at 5:40pm sharp.
Run at whatever pace is comfortable. Walk if you want.
Public holiday Mondays shift to Tuesday.
Christmas break dates are announced on Instagram each summer.
```

## Avoid

```text
No drop
Shot on Ricoh
Fabricated cancellation claims
Join our thriving community
The ultimate running experience
```

## SEO Without Sludge

Good:

```text
Beer Jerk Run Club is a free social run club in Auckland.
```

Bad:

```text
Looking for the best Auckland run club social running community running group Auckland? Join us.
```

## When Updating Pages

Each page should clearly answer:

- What is this?
- Who is it for?
- When and where is the run?
- What should a new person do next?

Keep facts visible in normal HTML text. Do not put key facts only inside images.
