Beer Jerk Run Club

Beer Jerk Run Club is our free Auckland run club with a big social side.

We meet every Monday at Small Gods Taproom in Eden Terrace. Bag drop is from 5pm and the run starts at 5:40pm sharp. The usual Monday run is roughly 5km, with a rotating set of routes around Eden Terrace, Maungawhau, Grange Road and the Domain when daylight allows.

No signup. No fees. No pace pressure. Run at whatever pace is comfortable, walk if you want, then stick around after.

$10 beers for runners after every run.

For the current schedule, routes, public holiday shifts and first-timer info, use the Beer Jerk Run Club site:
