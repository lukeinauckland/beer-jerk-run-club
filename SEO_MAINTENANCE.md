# SEO Maintenance

## Monthly

- Check Google Search Console for indexing issues.
- Check Bing Webmaster Tools for indexing issues.
- Confirm `/schedule` still shows useful upcoming dates.
- Confirm public holiday shifts are correct.
- Confirm Instagram and Strava links still work.
- Search branded terms:
  - `Beer Jerk Run Club`
  - `Beer Jerk Run Club Auckland`
  - `BJRC Auckland`

## Quarterly

- Check broad terms:
  - `Auckland run club`
  - `run club Auckland`
  - `social run club Auckland`
  - `Monday run club Auckland`
  - `Eden Terrace run club`
- Review whether Meetup is still worth paying for.
- Check image quality and replace weak photos.
- Ask for one or two relevant backlinks if natural:
  - Small Gods
  - Beer Jerk
  - Auckland running/event pages

## Seasonal

- Update Domain Loop status when daylight saving changes.
- Update Christmas break wording when dates are known.
- Add current Half Marathon section only when the next event is close enough and has a real ticket link.

## Before Every Deploy

Run:

```sh
npm run check
```

Do not deploy if it fails.

## What Success Looks Like

First:

- Branded terms rank.
- Google shows the correct title and description.
- The site appears when people search for Beer Jerk Run Club.

Then:

- The site starts appearing for local intent terms like `Monday run club Auckland` and `Eden Terrace run club`.

Harder:

- Broad terms like `Auckland run club` will need time, backlinks and consistent relevance.
